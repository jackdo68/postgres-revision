import { Card, CardGrid, LinkCard } from '@astrojs/starlight/components';

A hands-on PostgreSQL course for building schema-design instincts for production systems and interviews. Each week has a **lesson** (concepts + tradeoffs), **exercises** (hands-on SQL against the Pagila database), and **solutions** (answers + interview talking points).
